## 📊 技术栈总结

- **前端**: React 19, Next.js 15, TypeScript
- **UI库**: Ant Design 6
- **样式**: Tailwind CSS
- **数据库**: PostgreSQL
- **ORM**: Prisma 5
- **PDF**: jsPDF, html2canvas
- **部署**: Cloudflare Pages
- **日期处理**: dayjs
